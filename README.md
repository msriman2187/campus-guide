# Campus Guide 🎓

Campus Guide is an AI-powered campus information assistant built using Streamlit and a local LLM (Mistral via Ollama).

## 🚀 Features

- Automatically reads official college PDF documents
- Answers user questions using AI
- Clean premium dark UI
- Fully runs locally (no paid API required)

## 🛠 Tech Stack

- Python
- Streamlit
- Ollama (Mistral)
- PyMuPDF

## 🧠 How It Works

The system loads the college_data.pdf file, extracts text, and sends relevant context to a locally running LLM to generate intelligent responses.
